# 🏥 SSO Smart Planner (ประกันสังคมฉบับประชาชน)

โปรเจกต์วิเคราะห์ข้อมูลและวางแผนสวัสดิการประกันสังคมไทย เพื่อช่วยให้ผู้ประกันตนเข้าถึงสิทธิประโยชน์ได้ง่ายขึ้นและไม่เสียสิทธิ์เมื่อเปลี่ยนงาน

## ✨ ฟีเจอร์หลัก
1. **Dashboard วิเคราะห์เทรนด์:** ดูการเคลื่อนย้ายแรงงานรายจังหวัดย้อนหลัง 3 ปี (2565-2567)
2. **AI Advisor:** ปรึกษาปัญหากฎหมายประกันสังคมผ่านแชทบอทที่เข้าใจบริบทข้อมูลจริง
3. **Welfare Planner:** เครื่องมือคำนวณเงินบำนาญชราภาพแบบเข้าใจง่าย

## 🛠 เทคโนโลยีที่ใช้
- **Frontend:** Streamlit (Theme: Dark Mode)
- **Database:** DuckDB (ใช้ Query ไฟล์ Excel ความเร็วสูง)
- **Data:** Pandas & Plotly
- **AI:** Google Gemini API (หรือ OpenAI)

## 🚀 วิธีติดตั้ง
1. Clone repository: `https://github.com/Kaitokidbua/sso-project`
2. ติดตั้ง library: `pip install -r requirements.txt`
3. รันแอป: `streamlit run app.py`

## 📊 ข้อมูลที่ใช้อ้างอิง
- SSO Data Catalog (data.go.th) ปี 2565-2567

## Issues and motivation
Issues
1. แม้ว่าระบบประกันสังคมในปัจจุบันจะให้ข้อมูลและบริการพื้นฐานแก่ผู้ประกันตนได้ แต่ยังขาดเครื่องมือช่วยวางแผนเกษียณและคาดการณ์สิทธิประโยชน์ในอนาคต ทำให้ผู้ใช้งานไม่สามารถประเมินความพร้อมทางการเงินของตนเองได้อย่างมีประสิทธิภาพ
2. ข้อมูลสิทธิประโยชน์ของผู้ประกันตนในมาตรา 33, 39 และ 40 มีรายละเอียดและเงื่อนไขที่แตกต่างกัน ทำให้ผู้ใช้งานเข้าใจและเปรียบเทียบสิทธิของแต่ละมาตราได้ยาก
3. การค้นหาข้อมูลเกี่ยวกับสิทธิประโยชน์และเงื่อนไขต่าง ๆ ยังต้องอาศัยการอ่านเอกสารหรือค้นหาด้วยตนเอง ส่งผลให้ผู้ใช้งานใช้เวลามากและอาจได้รับข้อมูลไม่ครบถ้วน
Motivation
1. เพื่อพัฒนาเครื่องมือช่วยวางแผนเกษียณและคาดการณ์เงินบำนาญในอนาคต ช่วยให้ผู้ประกันตนสามารถเตรียมความพร้อมทางการเงินได้ดียิ่งขึ้น
2. เพื่อสร้าง Dashboard ที่สรุปและแสดงข้อมูลสิทธิประโยชน์ รวมถึงเปรียบเทียบความแตกต่างของมาตรา 33, 39 และ 40 ในรูปแบบที่เข้าใจง่าย
3. เพื่อพัฒนา AI Chatbot สำหรับตอบคำถามและให้ข้อมูลเกี่ยวกับประกันสังคมอย่างรวดเร็ว ช่วยลดเวลาในการค้นหาข้อมูลและเพิ่มความสะดวกในการเข้าถึงบริการของผู้ประกันตน
